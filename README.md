# Tour of Heroes with React Query

Inspired by Angular example app.

&nbsp;

## **Features**

1. list heroes from API.
1. create new heroes.
1. update hero name.
1. delete hero by id.
1. find hero by name.

&nbsp;

## **Technologies**

- React JS **18.2.0**
- React Query **4.20.4**
- React Router Dom **6.6.1**
- TypeScript **4.9.3**
- Vite JS **4.0.0**
- ReactStrap **9.2.0**
- Bootstrap **5.3.1**

&nbsp;

## **Installation**

1. Clone the repository

2.  Install dependencies of the project.

```shell
    npm install
```

3. Run API.
```shell
    npm run backend
```

4. Run the project.
```shell
    npm run dev
```
